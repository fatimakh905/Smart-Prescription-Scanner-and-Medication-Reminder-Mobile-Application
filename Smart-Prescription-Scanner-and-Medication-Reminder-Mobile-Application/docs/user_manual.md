## User Manual

### 1. Install the App
- Download APK file from the repository or release section.
- Install it on your Android device.
- Allow permissions (Camera, Notifications) when prompted.

---

### 2. Scan Prescription
- Open the app.
- Tap on ** + Add Prescription ** button.
- Capture image using camera or select from gallery.
- Wait while AI processes the prescription.

---

### 3. Verify Medicines
- After scanning, medicines will appear on verification screen.
- Check extracted details:
  - Medicine Name
  - Dosage
  - Frequency
  - Meal Instructions
  - Special Instructions
- Edit if required.
- Confirm to save medicines.

---

### 4. Manage Medicines
- Go to **Medicines Screen**.
- View all saved medicines.
- Add new medicine manually if needed.
- Edit or delete existing medicines.

---

### 5. Alarm Setup
- Alarms are generated automatically after saving medicines.
- User can adjust alarm time using time picker (like native clock app).
- Save alarms to activate reminders.

---

### 6. View Alarms
- Open **Alarms Screen**.
- View all scheduled alarms.
- See today's alarms automatically.

---

### 7. Get Notifications
- App will notify you at scheduled medicine times.
- Take medicine and follow instructions shown on screen.

---

### Notes
- Keep notifications enabled for proper alarm functioning.
- Ensure battery optimization is disabled for reliable alerts.
- Always verify prescription data before saving.
